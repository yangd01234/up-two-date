About Page

up2date is a single day supplemental Google calendar application.
To use the application, please use the following steps:

  1. Navigate to the homepage
  2. Create a new time or select an existing time
  3. Select the new time to start highlighting dates
  4. Select dates on the calendar
  5. Once you have selected one or more dates, you can click [Sign in to Google and Post]
  6. Login to your google account in the following screen
  7. Review the submission success message
  8. Congratulations, you posted to your calendar!
